Mern Stack

Front End

Html

  • Learn the basics: elements, tags, attributes.
  • Document structure: headings, paragraphs, lists, links.
  • Forms: input fields, buttons, form handling.
  • Media: images, audio, video embedding.

CSS

  • Basics: selectors, properties, box model.
  • Flexbox: flex container, flex items, alignment, spacing.
  • Grid: grid container, grid items, template areas.
  • Styling: colors, fonts, borders, backgrounds.

Project: Build Portfolio

  • Create a personal portfolio site with HTML and CSS.
  • Features: responsive layout, navigation bar, contact form.

JavaScript

  • Basics: variables, data types, operators, functions.
  • Control structures: conditionals, loops.
  • DOM manipulation: selecting elements, event listeners, modifying elements.
  • Events: handling user input, form submissions, clicks.

Project: Interactive JavaScript Project

  • Build a to-do list, calculator, or weather app.
  • Features: user interaction, DOM updates, dynamic content.

HTML & CSS Deep Dive

  • Semantic HTML: proper use of header, footer, main, article, section, etc.
  • Code indentation: maintain readable and maintainable code.
  • CSS animations & transitions: keyframes, easing functions, hover effects.
  • Media queries: responsive design techniques.
  • CSS frameworks: Sass or Tailwind.

Back End

Fetch

  • Learn how to use fetch for making HTTP requests.
  • Async programming: promises, async/await, error handling.
  • APIs: consuming RESTful APIs, handling JSON data.

Node.js

  • Introduction: Node environment, npm.
  • Express: setting up a server, routing, middleware.
  • File system: reading/writing files, serving static files.
  • APIs: building RESTful APIs with Express.

Authentication

  • Basics: session-based, token-based.
  • JWT: JSON Web Tokens, implementation.
  • OAuth: OAuth 2.0, integrating with Google, Facebook, etc.
  • Passport.js: middleware for authentication.

MongoDB

  • Basics: installing MongoDB, CRUD operations.
  • Data modeling: designing schemas, relationships between data.
  • Aggregation: filtering, grouping data.
  • Mongoose: using Mongoose for data modeling.

Project: Full-Stack Application

  • Build a CRUD application (e.g., blog, task manager).
  • Features: user authentication, data storage with MongoDB, API integration.

Advanced Topics

React.js

  • Basics: JSX, components, props, state.
  • Lifecycle methods: useEffect, useState.
  • Routing: React Router for navigation.
  • Forms: controlled components, form validation.
  • State management: Context API, Redux (optional).

TypeScript

  • Basics: types, interfaces, enums.
  • Advanced: generics, decorators, type inference.
  • Integration: adding TypeScript to JavaScript projects, using TypeScript with React.

Cloud

  • AWS basics: setting up an AWS account, S3, EC2.
  • Deployment: deploying applications with AWS services.
  • Docker: containerization basics, creating Docker images, running containers.

Web Security

  • Basics: HTTPS, CORS, secure headers.
  • Vulnerabilities: XSS, CSRF, SQL Injection.
  • Protection: input validation, authentication, authorization.

Server-Side Rendering (SSR)

  • Next.js basics: setting up a Next.js project, pages, and routing.
  • Rendering: static site generation (SSG), server-side rendering (SSR).
  • API routes: creating API endpoints with Next.js.

Testing

Jest

  • Basics: setup, writing tests.
  • Mocking: functions, modules.
  • Asserting: test results, snapshots.

React Testing Library

  • Basics: setting up, writing tests.
  • Component testing: rendering, querying, interacting with components.
  • User interactions: simulating clicks, typing, etc.
